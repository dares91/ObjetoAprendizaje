Formas de Propagación del Malware

El malware puede ser transferido de un usuario a otro y de un dispositivo a otro, mediante múltiples medios y mecanismos, entre ellos:

image El correo electrónico
image A través de software de mensajería instantánea como el messenger de Microsoft y servicios de chat proveídos por las redes sociales como facebook y google plus.
image Programas p2p como uTorrent, el cual es utilizado para compartir archivos.
image A través de la red inalámbrica
